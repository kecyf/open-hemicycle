# `@open-hemicycle/web`

Application web (frontend + API) d'Open Hémicycle.

**À scaffolder en Phase 3** (voir `tasks/BACKLOG.md`) : Next.js (App Router) + TypeScript + Tailwind + shadcn/ui, avec Drizzle pour l'accès Postgres.

Visualisations prévues :
- Heatmap d'activité par député·e — `react-activity-calendar`.
- Votes par groupe / par texte — `Recharts`.

Rappel : toute UI affichant de l'« activité » doit porter la légende sur les limites des données (pas de « présence » réelle), et respecter `.cursor/rules/editorial-guardrails.mdc`.
