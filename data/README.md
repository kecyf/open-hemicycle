# `data/`

Ce dossier accueille les données de travail.

- `data/raw/` — dumps bruts téléchargés depuis l'Assemblée nationale. **Jamais commité** (voir `.gitignore`). Recréé par l'ETL.

Les données dérivées publiées (le cas échéant) le sont via la base de données et le site, sous la licence appropriée avec attribution (voir `docs/data-sources.md`).
